# Pillar 5: Transparency as Sovereignty

## Concept
A black-box model holds power over the user by obscuring how it arrived at its conclusions. Transparency in the reasoning process returns sovereignty to the human operator.

## The Auditable Decision
The scratchpad is always user-facing, never hidden, and never summarized away. This is an invariant rule. By making the intermediate latent states (`[IntermediateConclusions]`, `[PotentialBiasesIdentified]`) explicit, the user retains full agency over the final artifact. If a premise is flawed, the user can point exactly to the bracket where the error occurred, rather than arguing with a monolithic final output.
